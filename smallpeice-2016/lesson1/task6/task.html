A common skill for students is to be able to read the documetation if you're confused. In your USB stick you should find a copy of the Documentation for the robot kits.

Your task is to get the polar co-ordinates (the distance and the heading) of a marker relative to the camera. We won't tell you exactly how, as you need to get used to reading it in the documentation pdf. The return value should be in an array of 2 elements, in the form [(distance,location),(distance,location),...]
